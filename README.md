# 予約ナビ

予約を忘れない、ナビも迷わない。あなたの予定をサポートするアプリです。

## 主な機能
- 📅 カレンダー連携で予定を自動取得
- 🔔 予約時刻に合わせて通知
- 🗺️ 通知からワンタップで Google Maps を起動
- 🛡️ 予定情報は端末内のみ利用、外部送信なし
